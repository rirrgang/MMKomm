Presentation

    Project Ideas & Goals:

  • Topic: Stock Market
  • Application consists of 3 parts:
    • 1. Breaking News line in the header
    • 2. UI for user input regarding personal preferences
    • 3. Personalized News output

    Project Challenges:

  • Filtering system for user preferences
  • Collecting multitple data sets from News-API

    Used Frameworks:

  • jQuery
  • News-API
  • nodejs
  • Vue.js
  • Bootstrap

    First Steps:

  • 1. Pulled & processed sample data from News-API
  • 2. Built a prototype with plain JS, CSS, HTML
  • 3. First design decisions made for Breaking News line

    Next Steps:

  • 1. Build filtering UI for user specified stock market news interests
  • 2. Offer user several filtering options via search bar & checkboxes
  • 3. Adjust news output to new user preferences
